## 2. 개발 환경

+ 개발 엔진 : Unity 2D

+ 언어 : C#

+ 형상 관리: SVN

<br>

## 3. 사용 기술
| 기술 | 설명 |
|:---:|:---|
| Sprite Animation | 마리오, 벽돌, 코인 애니메이션 |
| 타일맵 | 맵 구성 |
| Lerp | 카메라 팔로우 |
| Physics2D.Raycast | 포탈 이동 구현 |
